# Actividad Final Corte 1 - Python POO

Descripción:
Este proyecto implementa un sistema de vehículos inteligentes usando Programación Orientada a Objetos en Python.

Analogía:
Se utilizó la analogía de vehículos inteligentes que integran funcionalidades adicionales como GPS y sistema de audio.

Estructura:
- Clase abstracta: Vehiculo
- Clases adicionales: SistemaGPS, SistemaAudio
- Clase final: VehiculoInteligente (herencia múltiple)

Funcionalidades:
- Encendido de vehículos (polimorfismo)
- Visualización de ubicación
- Control de volumen
- Control de velocidad con encapsulamiento

Ejecución:
Para ejecutar el programa:

python main.py
